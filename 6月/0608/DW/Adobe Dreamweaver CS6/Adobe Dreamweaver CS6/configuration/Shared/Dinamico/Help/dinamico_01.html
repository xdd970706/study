Site-Specific Code Hints

Using the Site-Specific Code Hints interface you can create the configuration file needed to display code hints when working in Code View of Dreamweaver.

Note: The code hints are created specifically for the site selected in the Dreamweaver Files panel. In order for the code hints to display, the page you are working on must be saved to the currently selected site.

  1. To open the Site-Specific Code Hints interface, select Site > Site-Specific Code Hints.

    When first opened, the Site-Specific Code Hints dialog will scan your site to determine the framework that is being used. Dreamweaver includes 3 predefined frameworks that you can work with, Drupal, Joomla, and Wordpress.

  2. Specify the sub-root where your particular framework's files are stored.
  3. Choose the Add option to select a file or folder to be added to the scan. Within the Add Files/Folders interface, you will have the opportunity to select the file extensions to be included.
  4. Choose the Remove option if you wish to remove the selected files/folders from the scan.

    Note: If Drupal or Joomla is selected from the Structure menu, there will be an additional path included to a file within your Dreamweaver configuration folder. This is required when using these frameworks.

  5. To change how the Site-Specific Code Hints dialog treats a particular file/folder, select it from the list and do one of the following:
    • Scan this folder Select this option to have the selected directory included in the scan.
    • Recursive Will have all files and folders within the selected directory included in the scan.
    • Extensions Loads the Find Extensions dialog where you can view/edit any of the extensions included in your site structure.

The four buttons at the top of the dialog allow you to do the following.

  • Import Launches the import structure dialog for you to select a .config or XML file to use.
  • Save Launches the Save Structure Settings dialog.
  • Rename Launches the Rename Structure Settings dialog.
  • Delete Deletes the selected structure.

    Note: The default structures cannot be deleted.
